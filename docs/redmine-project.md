# Проект: Бот в МАХ для Zabbix-уведомлений

В рамках проекта планируется реализовать интеграцию Zabbix с ботом в МАХ для автоматической рассылки уведомлений о событиях мониторинга, требующих внимания сотрудников Отдела КБ.

**Цель проекта** — организовать доставку оперативных уведомлений из Zabbix в корпоративный мессенджер МАХ для сотрудников Отдела КБ, включая направления SOC и NOC.

На первом этапе проект ограничивается задачей доставки уведомлений из Zabbix. Бот используется как транспортный канал для оперативного информирования сотрудников Отдела КБ о событиях, инцидентах, сбоях и восстановлении сервисов, контролируемых в Zabbix.

## Входит в этап

- создание и настройка бота;
- настройка механизма отправки уведомлений из Zabbix в МАХ;
- определение формата уведомлений для SOC и NOC;
- маршрутизация уведомлений по группам или получателям;
- проверка доставки тестовых событий из Zabbix;
- фиксация минимальной эксплуатационной документации.

## Ожидаемый результат

Повысить надежность доставки оповещений системы мониторинга ответственным за сервисы до целевого уровня 100% за счет дублирования уведомлений в двух независимых каналах — Telegram и МАХ.
