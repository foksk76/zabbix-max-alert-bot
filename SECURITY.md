# Security policy

## Что считается чувствительными данными

Для этого проекта чувствительными считаются:

- значения авторизации;
- идентификаторы боевых чатов и пользователей;
- внутренние адреса систем мониторинга;
- реальные тексты внутренних инцидентов;
- названия закрытых организационных структур и внутренних проектов.

## Правило

В репозиторий добавляются только обезличенные примеры и шаблоны.

Если нужно показать настройку, использовать нейтральные значения:

```text
<bot-auth-value>
<chat-id>
<internal-zabbix-url>
```

## Сообщение о проблеме

Если в репозиторий случайно попали чувствительные данные:

1. не обсуждать их публично в issue;
2. удалить значение из текущей версии;
3. проверить историю Git;
4. заменить скомпрометированное значение в целевой системе.
