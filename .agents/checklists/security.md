# Security checklist

Минимальная проверка безопасности для этого репозитория.

## Нельзя хранить в репозитории

- реальные значения авторизации;
- идентификаторы боевых чатов и пользователей;
- внутренние адреса Zabbix и других систем;
- названия внутренних подразделений и закрытых проектов;
- выгрузки событий мониторинга с чувствительными данными.

## Можно хранить

- примеры с обезличенными значениями;
- шаблоны параметров;
- описание логики работы;
- тестовые сообщения без реальных данных;
- инструкции по настройке без публикации секретов.

## Проверка

Перед публикацией изменений просмотреть:

```text
README.md
docs/
examples/
src/
.agents/
.github/
```

Если есть сомнения, заменить конкретику на нейтральные формулировки: `получатель`, `чат`, `группа сопровождения`, `система мониторинга`, `внутренний адрес`.
